COMPILE_UNISIM
--------------------------------------------------------------------

Script to compile hdl projects with primitives from Xilinx following:
http://www.dossmatik.de/ghdl/ghdl_unisim_eng.pdf

Unfortunately does not work with *.vhd.

TO BE TESTED ON LINUX

COMPILE_UNISIM_AE

-a and -e flags
